# 3D-PLI-Demo

## install

### unix

``` sh
python3 -m venv env
source env/bin/activate
pip install pip --upgrade
pip install -r requirements.txt
python3 main.py
```

### windows

``` sh
python3 -m venv env
.\env\Scripts\activate
pip install pip --upgrade
pip install -r requirements.txt
python3 main.py
```
